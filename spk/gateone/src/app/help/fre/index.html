GateOne

Description

GateOne est un client SSH en HTML5. Il est capable de créer plusieurs connexions simultanées sur un ou plusieurs serveurs.

Site web de l'éditeur : liftoffsoftware.com

Paramétrage

Par défaut, GateOne écoute sur le port TCP 8022. Pour y accéder depuis internet, vous avez deux possibilités :

  • modifier la paramétrage du routeur principal
  • utilisant le paquet HAProxy qui permettra alors d'utiliser de multiplexer le port 443 du NAS vers différents services
